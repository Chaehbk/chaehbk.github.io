Furo i-Home

Summary

Furo i-Home is a voice-controlled home robot that connects with mobile phones and home appliances. I participated in voice user interface design and hardware design improvement at close-to-production phase.

Voice user interface design
Conducted multiple A/B testings to see how machine's voice-gender affects user perception depending on users' gender and task types

Hardware design improvement
Conducted lab-based experimental study on relationship between the screen orientation and user engagement over the video call to improve screen rotation setting

Methods

  • A/B testing
  • Analysis of variance(ANOVA)
